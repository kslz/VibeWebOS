## 执行摘要

MVP 27 条验收标准均已通过，部分 LLM/iframe 细节由自动化测试提供证据。无阻塞问题。

手工验收覆盖了桌面、四个图标、开始菜单、任务栏、多窗口、窗口最小化/最大化/关闭、应用搜索、候选应用生成、生成应用窗口、浏览器搜索与链接跳转、设置主题、关于系统、刷新清空状态、关闭窗口销毁状态。

自动化测试覆盖了 iframe 交互桥、生成应用本地控件不触发 LLM、标记控件触发 LLM、窗口级上下文隔离与销毁、HTML sanitizer、路由错误映射和 schema 校验。

## 已知限制

这些限制与第一版 non-goals 一致：

- 没有用户账号、数据库或持久化保存。
- 没有真实文件系统、真实下载、真实上传或真实网页访问。
- 浏览器是模拟浏览器，不访问真实网页，不提供多标签、收藏夹或复杂历史。
- 不支持壁纸切换、语言切换、插件系统、App 安装市场、多用户协作或移动端适配。
- 不提供真实登录、支付、转账、攻击、安全工具或隐私窃取能力。
- API Key 只允许由后端环境变量提供，前端不配置、不展示、不保存。
